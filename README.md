Application web conçu pour extraire des données à partir d'un fichier excel et l'enregistrer dans un BDD par oe biais d'un API web.
Api : PHP (Laravel)
Front-end : JavaScript (NextJs)

Installation et lancement :
-frontend :
cd frontend
npm install
npm run dev

-api :
cd api
composer require
php artisan serve
